# PRD: 익명 커뮤니티 게시판 "소문의 광장"

## 1. 프로젝트 개요

### 1.1 프로젝트명
**소문의 광장** - 익명 커뮤니티 게시판

### 1.2 목적
로그인 없이 누구나 자유롭게 가십, 소문, 신기한 이야기를 공유할 수 있는 가벼운 커뮤니티 플랫폼

### 1.3 기술 스택
- **Frontend**: React + TypeScript + Vite
- **UI Framework**: Mantine UI v7
- **Icons**: Bootstrap Icons
- **Backend**: Supabase (추후 연동)
- **상태관리**: React useState/useContext (간단하게)

---

## 2. 핵심 기능

### 2.1 카테고리 구조
| 카테고리 | 설명 | 아이콘 |
|---------|------|--------|
| 세상의 가십거리 | 연예인, 유명인 관련 소문 | bi-chat-quote |
| 카더라통신 | 확인되지 않은 소문, 루머 | bi-broadcast |
| 비밀이야기 | 신기하고 놀라운 이야기 | bi-emoji-surprise |

### 2.2 게시글 기능
- ✅ 익명 글 작성 (닉네임 선택 입력, 미입력시 "익명")
- ✅ 제목 + 본문 작성
- ✅ 카테고리 선택
- ✅ 작성 시간 표시
- ✅ 조회수 표시
- ✅ 댓글 수 표시

### 2.3 댓글 기능
- ✅ 익명 댓글 작성
- ✅ 댓글 시간 표시
- ✅ 대댓글 없음 (심플하게)

### 2.4 목록/상세 기능
- ✅ 카테고리별 필터링
- ✅ 전체 글 목록 (최신순)
- ✅ 무한 스크롤 또는 페이지네이션
- ✅ 게시글 상세 보기

## 4. 데이터 모델 (Supabase 연동용)

### 4.1 posts 테이블
```typescript
interface Post {
  id: string;
  category: 'gossip' | 'rumor' | 'amazing';
  nickname: string;
  title: string;
  content: string;
  views: number;
  created_at: string;
}
```

### 4.2 comments 테이블
```typescript
interface Comment {
  id: string;
  post_id: string;
  nickname: string;
  content: string;
  created_at: string;
}
```
